# 📊 Business Intelligence Dashboards

A collection of Business Intelligence (BI) projects featuring interactive dashboards built with **Microsoft Power BI** and **Tableau** for data-driven decision making.

---

**Tools & Technologies:**
- Microsoft Power BI
- DAX (Data Analysis Expressions)
- Power Query (ETL)
- Supermarket Sales Dataset

---

## 🛠️ Tools & Technologies Used

| Tool | Purpose |
|------|---------|
| Microsoft Power BI | Dashboard creation & visualization |
| Tableau | Data visualization & analytics |
| DAX | Custom measures & calculated columns |
| Power Query | Data transformation & cleaning |
| Excel / CSV | Data source |

---

## 🚀 Getting Started

### Prerequisites
- [Microsoft Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free download)
- [Tableau Public](https://public.tableau.com/) or Tableau Desktop

### How to Use
1. Clone this repository:
```bash
   git clone https://github.com/SanaullahTareen/Bussiness_Intelligence_BI_Tableau.git
```
2. Navigate to the project folder of your choice.
3. Open the `.pbix` file in **Power BI Desktop** or the `.twbx` file in **Tableau**.
4. Explore the interactive dashboards.
